Gerar Arquivo Excel Anual

Clique no botão abaixo para gerar um arquivo Excel com planilhas para cada mês do ano, utilizando os dados salvos.


Os dados serão exportados em formato de planilha, com as seguintes colunas:

  • DIA: Dia do mês
  • DESCRIÇÃO: Descrição da transação
  • CRÉDITO + : Valor creditado
  • DÉBITO - : Valor debitado
  • SALDO DO DIA: Saldo acumulado até o dia

Os dados serão exportados para um arquivo Excel com planilhas separadas para cada mês do ano.
